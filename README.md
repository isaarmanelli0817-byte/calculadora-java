# Calculadora em Java

Projeto desenvolvido no 1º período do curso de Análise e Desenvolvimento de Sistemas.

## Sobre o projeto
Uma calculadora simples no console que realiza as 4 operações básicas: soma, subtração, multiplicação e divisão.

## O que eu aprendi
- Entrada de dados com Scanner
- Estruturas de condição if/else
- Operações matemáticas em Java
- Organização de código

## Como executar
1. Abra o projeto em uma IDE como IntelliJ ou Eclipse
2. Execute o arquivo `Calculadora.java`
3. Siga as instruções no console

## Tecnologias
- Java
